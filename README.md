# A股联网估值模型

这是一个纯静态网站版本，入口文件是 `index.html`。网站可以部署到 GitHub Pages、Netlify、Vercel 或任意静态服务器。

## 已优化内容

- 去掉微信文件路径依赖，整理成独立静态网站工程。
- 保留输入股票代码或公司名称自动联网取数。
- 覆盖沪市、深市、创业板、科创板、北交所常见代码格式。
- 增加移动端布局、深色模式、分享链接、历史记录导出。
- 增加 GitHub Pages Actions 配置。
- 买入系数中的负债修正项加入上下限保护，降低极端负债率对估值的扭曲。
- 增加投行估值买卖点：DCF/FCFF、远期 PE、FCF 收益率三种口径加权，并输出投行合理市值、买点、卖点和对应股价。
- 当 N+2 自由现金流为负时，总体盈余口径显示为不适用，不再输出负卖点，也不纳入综合卖点均值。
- 当当前市值高于综合卖点时，页面会明确显示高于卖点的幅度，并标记为模型提示高估，而不是把卖点强行抬到当前市值上方。
- 盈利预测优先使用东方财富一致预测；若结构化一致预测为空，会继续尝试东方财富 F10 与同花顺 F10 研报摘要；券商预测仍缺失时，结合行业属性、经营修复、资本开支、中长期股价走势、市场确认度和热点属性生成投行方式简易盈利预测，并在近三年利润图表中标注预测来源。
- 对券商预测仍未转正、归母净利润或 PE 为负的公司，PE 口径不作为核心估值，转用收入和正常化利润率的非 PE 口径辅助判断。

## 本地预览

```bash
npm run start
```

打开：

```text
http://127.0.0.1:4173/
```

## 发布到 GitHub Pages

把本目录推送到 GitHub 仓库的 `main` 分支后，仓库的 Pages 使用 GitHub Actions 发布即可。工作流文件已经放在 `.github/workflows/pages.yml`。

发布后可用：

```text
https://<你的GitHub用户名>.github.io/<仓库名>/
```

## 数据源说明

- 实时行情、股息率TTM、总股本、行业、毛利率和负债率：东方财富公开接口。
- 财务报表：东方财富数据中心公开接口。
- D&A：同花顺年度现金流量表“固定资产折旧、油气资产折耗、生产性生物资产折旧 + 无形资产摊销 + 长期待摊费用摊销”，优先取近三年明细，并结合近年趋势做上调或下调。
- Capex：同花顺年度现金流量表“购建固定资产、无形资产和其他长期资产支付的现金”，优先取近三年，并结合近年投资趋势做上调或下调。
- 盈利预测：优先使用东方财富券商盈利预测公开接口；若结构化预测为空，则补充东方财富 F10 与同花顺 F10 研报摘要；仍取不到时，按收入趋势、利润修复、净利率回归、行业属性、资本开支压力、中长期股价走势和市场确认度生成投行方式简易盈利预测。
- 止盈系数：沪深300或中证500日线 MA60 / MA250。
- 投行估值口径：使用未来五年 FCFF 折现 + 终值、N+2 远期 PE、N+2 FCF 收益率三种方法加权；安全边际按负债率、股息率、预期增速和毛利率变化动态调整。
- 综合卖点：只平均正数且有效的卖点口径；若自由现金流为负，总体盈余卖点作为风险提示，不作为卖点价格输入。
- 卖点低于当前市值时，表示当前市场价格已经超过模型给出的卖点，不代表计算应自动上调。

## 注意

公开接口可能受限流、跨域策略或上游字段变化影响。网站已做降级处理，但结果仍只适合研究参考，不构成投资建议。
